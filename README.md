# Practica 3 - Sistemas Inteligentes - Algoritmos Geneticos
--------------------------------------------------------

# Descripción:
Este repositorio fue creado para usos educativos para la materia de Sistemas Inteligentes. 
El objetivo de la practica es implemetar un Algoritmo Genetico. En este caso se utilizo el ejemplo mencionado en el libro
de "http://natureofcode.com/" en la sección de "Evolving Forces: Smart Rockets".

# Librerias
Para este proyecto se utiliza la libreria de p5js.
- https://p5js.org

# Uso
1. Descargar/clonar el repositorio (Este repositorio ya contiene la librerias necesarias para correr)
2. Ejecutar index.html
